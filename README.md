# Heart Disease Prediction (Machine Learning)

Projeto educacional de ML para prever a probabilidade de doença cardíaca.

## Objetivo
Construir um modelo capaz de estimar o risco de doença cardíaca.

## Modelo
- KNN (modelo final escolhido) -> O modelo RF não foi escolhido pois em testes comparativos, apresentou a menor taxa de acerto (accuracy) no conjunto de teste em relação a outros modelos avaliados.

## Funcionalidades
- Pré-processamento com Pipeline
- Treinamento e avaliação do modelo
- Predição de novos pacientes via entrada do usuário
- Exibição de probabilidade e classificação de risco

## Aviso
Este projeto é apenas educacional e não deve ser usado para fins médicos.
